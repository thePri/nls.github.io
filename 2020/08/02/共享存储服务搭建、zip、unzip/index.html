安装zip和unzip

  1. 查看系统版本

  2. 下载zip、unzip的rpm包，下载地址zip、unzip下载

  3. 将压缩包传到Linux服务器

  4. 安装zip

    –force: 强制执行

    –nodeps: 不查找依赖关系

  5. 安装unzip

    –force: 强制执行

    –nodeps: 不查找依赖关系

  2. 将rpm包上传至服务器

  3. 安装nfs

  4. nfs配置

  5. 设置开机启动

  6. 启动相关服务
